# 三、携手同行之 -- 定制命令提示符

命令提示符由一个名为PS1的shell变量定义。我们可以用set命令打印所有的shell变量，然后用 / 搜索到PS1变量。
也可以直接用 echo $PS1 打印出PS1的变量值。  
在修改命令提示符之前，我们先对原来的值进行备份：ps1_old=$PS1  
备份完成就可以大胆的进行修改了，格式为：PS1="..."，将要修改的内容写在双引号内。我们可以自定义命令提示符的内容、文字颜色、背景颜色。

例如：PS1="<\u@\h \W>\$" 表示命令符包含：用户名、主机名、@、最后打开的目录和一个$，它的显示效果如下：
<yeqing@LAB ~>$。  
我们还可以设置颜色，比如：PS1="\033[0;32m<\u@\h \W>\$\033[0m"，这样显示的命令符就会是绿色的。

*注意：如果设置了颜色，最后一个要关闭颜色，否则你在提示符后面输入的命令也将会是设置的颜色，关闭颜色的转义字符为\033[0m*

如果想让设置的自定义命令提示符永久保存，可以在~/.bashrc或~/.bash_profile文件中设置PS1的值。
